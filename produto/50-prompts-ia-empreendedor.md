# 50 Prompts de IA para Empreendedor
## Kit de Produtividade com Inteligência Artificial

---

## Como usar este kit

1. Copie o prompt desejado
2. Cole no ChatGPT, Claude ou Gemini (todos gratuitos)
3. Substitua o que está entre [colchetes] pelas suas informações
4. Pronto — resultado profissional em segundos

---

## Categoria 1: Vendas e Conversão (Prompts 1-10)

### Prompt 1 — Copy de anúncio Meta Ads

```
Crie um anúncio para Facebook/Instagram sobre [produto/serviço].
Público: [descreva o público].
Dor principal: [qual problema resolve].
Diferencial: [seu diferencial].
Tom: persuasivo e direto.
Formato: texto principal + headline + CTA.
Inclua 3 variações para teste A/B.
```

### Prompt 2 — Sequência de e-mails para follow-up

```
Crie uma sequência de 4 e-mails para follow-up de leads que
baixaram meu material gratuito sobre [tema].
Lead magnet: [nome do material].
Produto principal: [produto] por [preço].
Tom: educativo e depois persuasivo.
Objetivo: converter em compra até o 4o e-mail.
Inclua assunto, corpo do e-mail e CTA de cada um.
```

### Prompt 3 — Script de VSL (Video Sales Letter)

```
Escreva um roteiro de VSL de 3 minutos para [produto/serviço].
Abertura: ganhe atenção nos primeiros 5 segundos.
Problema: descreva a dor principal do [público-alvo].
Solução: apresente meu produto como resposta.
Prova: inclua um case ou depoimento.
Oferta: preço de [valor] com [garantia/diferencial].
CTA: compre agora.
Tom: conversacional, direto ao ponto.
```

### Prompt 4 — Oferta irresistível

```
Monte uma estrutura de oferta para [produto/serviço] de
[preço original] que gere urgência e escassez.
Inclua: bônus, garantia, prazo limitado, desconto.
Público: [descrição do cliente ideal].
Objetivo: aumentar conversão em 30%.
```

### Prompt 5 — Perguntas para qualificar lead no WhatsApp

```
Crie um roteiro de 5 perguntas para qualificar leads no WhatsApp
que responderam a [anúncio/publicação].
Produto: [produto].
Objetivo: identificar dor, orçamento e urgência em 3 minutos
de conversa.
Tom: consultivo, não vendedor.
```

### Prompt 6 — Resposta para objeção de preço

```
Me dê 5 respostas prontas para a objeção "é muito caro" para
meu produto [nome] de [preço].
Inclua em cada resposta: reenquadramento de valor,
comparação com custo de não ter o produto, e prova social.
Tom: educado, confiante, sem ser defensivo.
```

### Prompt 7 — Texto para WhatsApp de vendas

```
Escreva um texto de vendas para WhatsApp com 200-300 caracteres
para [produto/serviço]. Inclua: saudação personalizada,
dor/problema, solução, CTA com link.
Tom: conversacional, próximo, sem ser invasivo.
```

### Prompt 8 — Proposta comercial por mensagem

```
Crie uma proposta comercial enxuta para enviar por mensagem
(Direct/WhatsApp) para [cliente ideal] sobre [produto/serviço].
Diferencial: [seu diferencial].
Preço: [valor].
Inclua: problema, solução, resultado, investimento, CTA.
Máximo 400 caracteres.
```

### Prompt 9 — Script de abordagem para prospecção

```
Crie 3 scripts de abordagem para prospecção ativa no Instagram
Direct para vender [produto/serviço].
Público: [descreva].
Abordagem 1: elogio + problema + oferta.
Abordagem 2: pergunta + solução.
Abordagem 3: case + convite.
Tom: útil, não spam.
```

### Prompt 10 — Gatilhos mentais para descrição de produto

```
Reescreva a descrição abaixo do meu produto [cole a descrição]
aplicando estes gatilhos mentais em cada parágrafo:
1. Prova social (pessoas usam)
2. Autoridade (especialista criou)
3. Urgência (tempo limitado)
4. Especificidade (dados exatos)
5. Reciprocidade (bônus)
Formato: descrição final pronta para Hotmart/Kiwify/Gumroad.
```

## Categoria 2: Instagram e Redes Sociais (Prompts 11-20)

### Prompt 11 — Calendário de conteúdo mensal

```
Crie um calendário de conteúdo para Instagram de 30 dias para
[meu nicho/negócio].
Distribuição: 50% educativo, 20% entretenimento, 20% vendas,
10% bastidores.
Formato: data + tema + formato (post/carreosel/reels) + legenda curta.
Público: [descreva].
```

### Prompt 12 — Legenda para post de venda

```
Escreva 5 legendas para post de venda no Instagram sobre
[produto/serviço].
Cada legenda: 100-200 caracteres.
Inclua em cada: hook nos primeiros 2 segundos, dor, solução,
CTA.
Variações: tom direto, tom故事, tom perguntador.
```

### Prompt 13 — Roteiro de Reels de 30 segundos

```
Crie 3 roteiros de Reels de 30 segundos para divulgar
[produto/serviço].
Estrutura de cada: hook (3s) + problema (5s) + solução (10s) +
demonstração (7s) + CTA (5s).
Inclua sugestão de texto na tela e música (estilo).
```

### Prompt 14 — Responder comentários e DMs

```
Crie 10 respostas prontas para comentários e DMs no Instagram
sobre [produto/serviço]. Inclua respostas para:
1. "Quanto custa?"
2. "Como funciona?"
3. "Tem garantia?"
4. "Indica para iniciantes?"
5. "Tem desconto?"
6. "Funciona para [caso específico]?"
7. "Pode parcelar?"
8. "Tem suporte?"
9. "Vou pensar"
10. "Já comprei, como acessa?"
Tom: acolhedor, direto, profissional.
```

### Prompt 15 — Bio do Instagram que converte

```
Escreva 3 opções de bio do Instagram para meu negócio de
[nicho].
Incluir em cada: quem sou, o que faço, resultado entrego,
CTA (link).
Máximo 150 caracteres cada.
Diferencial: [seu diferencial].
```

### Prompt 16 — Chamada para ação (CTA) criativa

```
Crie 20 CTAs criativos para posts de Instagram sobre
[produto/serviço]. Variações:
5 CTAs diretos ("Compre agora")
5 CTAs curiosidade ("Quer saber como?")
5 CTAs escassez ("Últimas vagas")
5 CTAS engajamento ("Me diz nos comentários")
```

### Prompt 17 — Ideias de stories diários

```
Gere 30 ideias de stories diários para [meu nicho] que
engajem e vendam. Distribuição:
10 stories educativos
5 stories de bastidores
5 stories de provas sociais
5 stories interativos (enquetes, perguntas)
5 stories de vendas
Descreva cada ideia em 1 linha.
```

### Prompt 18 — Post de autoridade

```
Escreva um post de autoridade para Instagram sobre [tema que
domino]. Estrutura: minha experiência + dados do setor +
ensinamento prático + CTA sutil.
Tom: confiante, generoso.
Público: [descreva].
```

### Prompt 19 — Thread no Twitter/LinkedIn

```
Crie uma thread de 5-8 posts para LinkedIn ou Twitter sobre
[tema relevante do meu nicho].
Post 1: gancho (pergunta ou dado impactante)
Posts 2-6: ensinamentos em sequência
Post 7: resumo
Post 8: CTA
Tom: profissional, mas acessível.
```

### Prompt 20 — Hashtags estratégicas

```
Me dê um pacote de 30 hashtags para Instagram segmentadas
para [meu nicho/negócio], divididas em:
10 hashtags de volume (500k+ posts)
10 hashtags médias (50k-500k)
10 hashtags de nicho (menos de 50k)
Inclua também 5 variações de pacotes para diferentes temas
dentro do meu nicho.
```

## Categoria 3: Atendimento e Suporte (Prompts 21-30)

### Prompt 21 — Resposta para reclamação

```
Crie 5 respostas para reclamação de cliente sobre [problema
comum no meu negócio]. Tom: empático, resolutivo.
Estrutura: 1) validar sentimento 2) pedir desculpas 3) explicar
causa 4) oferecer solução 5) compensação se aplicável.
```

### Prompt 22 — FAQ completo

```
Crie um FAQ com 15 perguntas e respostas sobre [meu
produto/serviço]. Inclua perguntas sobre: preço, entrega,
garantia, suporte, funcionamento, devolução, diferenciais.
Tom: claro, direto, tranquilizador.
```

### Prompt 23 — Mensagem de pós-venda

```
Escreva 3 mensagens de pós-venda para enviar 24h após a
compra de [produto/serviço].
Mensagem 1: boas-vindas e agradecimento
Mensagem 2: instruções de uso
Mensagem 3: pedido de review/indicação
Tom: acolhedor, generoso.
```

### Prompt 24 — Roteiro de onboarding

```
Crie um roteiro de onboarding em 5 passos para novos clientes
de [produto/serviço]. Cada passo: o que fazer + instrução
clara + resultado esperado. Formato: mensagens curtas para
enviar por WhatsApp ou e-mail.
```

### Prompt 25 — Lidar com cliente difícil

```
Crie 5 respostas profissionais para lidar com cliente difícil
no WhatsApp sobre [situação comum]. Situações:
1. Cliente exigindo desconto após compra
2. Cliente insatisfeito com prazo
3. Cliente com dúvida repetitiva
4. Cliente agressivo
5. Cliente querendo cancelar
Em cada: resposta pronta + princípio de atendimento.
```

### Prompt 26 — Pesquisa de satisfação

```
Crie uma pesquisa de satisfação de 5 perguntas para enviar
após a compra de [produto/serviço]. Misture: múltipla escolha
+ pergunta aberta. Inclua introdução acolhedora e
agradecimento. Objetivo: NPS + insights de melhoria.
```

### Prompt 27 — E-mail de recuperação de carrinho

```
Crie uma sequência de 3 e-mails para recuperar carrinho
abandonado de [produto/serviço] de [preço].
E-mail 1: lembrete + benefícios (enviar 1h após abandono)
E-mail 2: objeções + prova social (enviar 24h depois)
E-mail 3: urgência + bônus exclusivo (enviar 48h depois)
Assuntos: persuasivos sem ser apelativos.
```

### Prompt 28 — Mensagem de aniversário para cliente

```
Crie 3 modelos de mensagem de aniversário para cliente com
oferta especial de [produto/serviço]. Inclua: cumprimento
personalizado + presente/desconto + prazo de uso + CTA.
Tom: caloroso, celebratório.
```

### Prompt 29 — Roteiro de suporte técnico básico

```
Crie um roteiro de diagnóstico rápido para suporte técnico
de [produto digital/serviço]. Passos: 1) identificar problema
2) perguntas de diagnóstico 3) soluções comuns 4) escalonamento.
Formato: árvore de decisão simples.
```

### Prompt 30 — Mensagem de reactivação

```
Escreva 5 mensagens para reativar clientes inativos há mais
de 90 dias do meu [produto/serviço].
Estratégias: novidade, desconto exclusivo, conteúdo gratuito,
prova social de novos resultados, medo de perder.
Cada mensagem: 100-150 caracteres.
```

## Categoria 4: Finanças e Gestão (Prompts 31-40)

### Prompt 31 — Precificação de produto/serviço

```
Ajude-me a precificar [produto/serviço]. Custos: [liste].
Mercado: concorrentes cobram entre [X] e [Y].
Público: [classe social/profissão].
Valor entregue: [descreva resultado].
Me dê 3 opções de preço: entrada, ideal, premium.
Para cada: justificativa + margem estimada.
```

### Prompt 32 — Planejamento financeiro mensal

```
Crie um roteiro de planejamento financeiro mensal para
[meu tipo de negócio]. Inclua: 1) projeção de receita
2) custos fixos 3) custos variáveis 4) margem 5) metas.
Formato: check-list + planilha mental.
```

### Prompt 33 — Análise de lucro por produto

```
Ajude-me a calcular a lucratividade real de [produto/serviço]
considerando: preço de venda [X], taxa da plataforma [Y]%,
custo de aquisição [Z], frete/entrega [W], impostos [V]%.
Me dê: lucro bruto, lucro líquido, margem %, ponto de
equilíbrio em unidades.
```

### Prompt 34 — Controle de fluxo de caixa

```
Crie uma planilha conceitual de fluxo de caixa para pequeno
negócio. Colunas: data, descrição, entrada, saída, saldo,
categoria. Linhas: 30 dias. Inclua categorias comuns:
vendas, plataforma, marketing, material, fixas, retirada.
```

### Prompt 35 — Decisão: investir ou não

```
Ajude-me a decidir se devo investir [valor] em [ação:
exemplo: anúncios pagos, novo produto, ferramenta].
Cenário atual: [descreva].
Retorno esperado: [X].
Risco: [alto/médio/baixo].
Me dê: análise custo-benefício, timeline de retorno,
alternativas mais baratas, decisão recomendada.
```

### Prompt 36 — Meta de faturamento e desdobramento

```
Defina metas realistas de faturamento para meu [tipo de
negócio] nos próximos 3 meses. Meta mensal: [X].
Desdobre em: quantas vendas/dia, ticket médio necessário,
leads necessários, taxa de conversão requerida.
Considere: minha realidade atual de [Y] vendas/mês.
```

### Prompt 37 — Controle de despesas

```
Crie 5 categorias de despesas para meu negócio digital e
sugira um orçamento mensal para cada. Receita estimada: [X].
Categorias: 1) operação 2) marketing 3) ferramentas
4) desenvolvimento 5) reserva.
Para cada: % ideal do faturamento e valor em R$.
```

### Prompt 38 — Roteiro de revisão financeira semanal

```
Crie um roteiro de 15 minutos para revisão financeira semanal:
Passo 1: [o que verificar]
Passo 2: [o que analisar]
Passo 3: [o que decidir]
Inclua perguntas-chave para cada passo.
Objetivo: manter saúde financeira com mínimo esforço.
```

### Prompt 39 — Comparação de plataformas de venda

```
Compare as plataformas [Hotmart, Gumroad, Kiwify] para vender
[meu tipo de produto digital]. Critérios: taxa, público,
facilidade, pagamento BR, suporte, ferramentas.
Conclusão: qual é a melhor para o meu caso e por quê.
```

### Prompt 40 — Precificação de mentoria/consultoria

```
Ajude-me a precificar uma mentoria/consultoria de [tema].
Minha experiência: [X anos/resultados].
Formato: [online/presencial, quantas sessões].
Concorrência: [faixa de preço].
Valor entregue: [resultado específico].
Sugira 3 níveis de preço e o que incluir em cada.
```

## Categoria 5: Produtividade e Organização (Prompts 41-50)

### Prompt 41 — Planejamento semanal ideal

```
Crie meu planejamento semanal ideal para [tipo de negócio].
Distribuição: 30% produção, 20% vendas, 20% marketing,
15% gestão, 15% aprendizado.
Segunda a sexta: blocos de 2h com tarefa específica.
Inclua pausas e horário de início/término realistas.
```

### Prompt 42 — Priorização de tarefas (Matriz Eisenhower)

```
Ajude-me a priorizar minhas tarefas desta semana usando a
Matriz Eisenhower (urgente x importante). Minha lista:
[cole suas tarefas]. Classifique cada uma e me dê um plano
de ação para a semana com as 3 tarefas mais importantes.
```

### Prompt 43 — Sistema de organização de arquivos

```
Crie uma estrutura de pastas organizada para meu negócio
digital. Pastas principais:
1. Produto 2. Marketing 3. Vendas 4. Financeiro 5. Clientes
6. Aprendizado
Subpastas recomendadas para cada. Formato: árvore.
```

### Prompt 44 — Checklist de lançamento de produto

```
Crie um checklist completo de lançamento de produto digital
com 30 itens, dividido em fases:
Pré-lançamento (D-30 a D-8): [itens]
Aquecimento (D-7 a D-1): [itens]
Lançamento (D0 a D7): [itens]
Pós-lançamento (D8 a D30): [itens]
Cada item: descrição + status (a fazer/fazendo/pronto).
```

### Prompt 45 — Rotina de criação de conteúdo

```
Desenhe uma rotina de criação de conteúdo que produza
7 dias de posts em 2 horas. Inclua:
1. Pesquisa (20 min)
2. Criação em lote (60 min)
3. Design (25 min)
4. Agendamento (15 min)
Ferramentas sugeridas: gratuitas.
```

### Prompt 46 — Delegação e terceirização

```
Liste 10 tarefas do meu negócio que posso delegar ou
automatizar. Para cada: tarefa, ferramenta de automação
sugerida, tempo economizado/semana, prioridade (alta/média).
Considere: [meu nicho].
```

### Prompt 47 — Reunião de 15 minutos

```
Crie uma pauta para reunião de alinhamento semanal de
15 minutos com [equipe/parceiro/fornecedor].
Pauta:
1. [O que fazer?]
2. [O que está bloqueado?]
3. [O que aprendemos?]
4. [Próximos passos?]
Regra: 3 min por tópico, sem exceção.
```

### Prompt 48 — Sistema de metas e hábitos

```
Crie um sistema simples de 3 metas e 3 hábitos diários para
fazer meu negócio crescer este mês.
Cada meta: específica, mensurável, prazo.
Cada hábito: ação concreta de <15 minutos.
Formato: 1 tabela simples para imprimir e colar na parede.
```

### Prompt 49 — Análise semanal de resultados

```
Crie um template de análise semanal de resultados com:
Métricas: vendas, leads, tráfego, conversão, receita, custos.
Perguntas: o que funcionou, o que não funcionou, o que
vou fazer diferente.
Formato: 5 minutos para preencher.
```

### Prompt 50 — Plano de 90 dias

```
Crie um plano de 90 dias para [objetivo específico do meu
negócio]. Divida em:
Mês 1 (dias 1-30): [foco + ações + meta]
Mês 2 (dias 31-60): [foco + ações + meta]
Mês 3 (dias 61-90): [foco + ações + meta]
Inclua 1 indicador principal para cada mês.
```

## BÔNUS SECRETO: 5 Prompts de Alta Conversão

### Bônus 1 — Copy de lançamento completa

```
Crie a copy completa de lançamento para [produto] de [preço].
Estrutura:
1. Headline (promessa forte, <15 palavras)
2. Subheadline (especificidade)
3. Dor e agitação (2 parágrafos)
4. Solução (apresentação do produto)
5. O que está incluído (bullet points)
6. Prova social (depoimento)
7. Oferta + bônus (valor percebido)
8. Garantia (reversão de risco)
9. CTA final (ação imediata)
10. PS (último gancho)
Tom: persuasivo, específico, urgente.
```

### Bônus 2 — Script de vendas em 5 minutos

```
Crie um script de vendas de 5 minutos para [produto] de
[preço] em ligação ou chamada de vídeo.
Minuto 1: rapport + descoberta da dor
Minuto 2: apresentação da solução
Minuto 3: objeções + prova social
Minuto 4: oferta + bônus + garantia
Minuto 5: fechamento + CTA
Para cada minuto: o que falar + perguntas-chave.
```

### Bônus 3 — Sequência de lançamento para Instagram

```
Crie uma sequência de 7 dias de posts para aquecimento de
lançamento de [produto] de [preço].
Dia 1: provocação (gancho)
Dia 2: problema (dor específica)
Dia 3: educação (valor gratuito)
Dia 4: bastidores (prova de criação)
Dia 5: pré-venda (early adopter)
Dia 6: venda (abertura)
Dia 7: urgência (fechamento + bônus)
Para cada dia: formato + legenda + CTA.
```

### Bônus 4 — E-mail de vendas de alto ticket

```
Crie um e-mail de vendas para [produto/serviço de alto
ticket] (acima de R$500).
Assunto: personalizado + curiosidade.
Corpo: case de sucesso (específico, com números) + processo
+ resultado + oferta + garantia estendida.
CTA: agendar call ou comprar agora.
Tom: consultivo, confidente, sem pressão.
```

### Bônus 5 — Copy para página de obrigado

```
Crie uma copy para página de obrigado pós-compra de [produto].
Inclua: entusiasmo, instruções claras de acesso,
entrega de bônus, próximo passo (consumir conteúdo),
convite para comunidade/suporte.
Tom: celebratório, acolhedor.
```
