## Categoria 1: E-mail Marketing (Prompts 51-60)

### Prompt 51 — Newsletter semanal
Crie uma newsletter semanal de [tema] com 3 seções:
1) Insight rápido (1 parágrafo) 2) Dica prática (passo a passo)
3) Recurso recomendado (ferramenta/livro). Tom: próximo, educativo.
Público: [descreva].

### Prompt 52 — Sequência de nurture (5 e-mails)
Crie uma sequência de 5 e-mails de nurture para leads que
baixaram [material gratuito]. E-mail 1: entrega + expectativa.
E-mail 2: dica complementar. E-mail 3: case de sucesso.
E-mail 4: nova perspectiva. E-mail 5: oferta + CTA.
Objetivo: educar antes de vender.

### Prompt 53 — E-mail de reengajamento
Escreva um e-mail para reengajar assinantes que não abrem
há 60+ dias. Assunto: intrigante, pessoal. Corpo: valor
gratuito + novidade + CTA fraco (sem venda). Objetivo:
recuperar atenção, não vender.

### Prompt 54 — Copy de e-mail de vendas diretas
Crie um e-mail de vendas diretas para [produto] de [preço].
Assunto: curiosidade + benefício. Pré-header: complemento.
Corpo: dor → solução → prova → oferta → urgência → CTA.
Tom: confiante, direto. Máximo: 300 palavras.

### Prompt 55 — E-mail de lançamento (Dia 1)
Escreva o e-mail de abertura de lançamento para [produto].
Assunto: grande anúncio. Corpo: expectativa criada nos
pré-lançamentos → anúncio oficial → o que muda → bônus
por tempo limitado → CTA. Tom: comemorativo, urgente.

### Prompt 56 — E-mail de quebra de objeção
Crie um e-mail que responda às 3 principais objeções de
[produto]: preço, tempo, resultado. Para cada: 1) validação
2) reenquadramento 3) contra-prova. CTA: link de compra.

### Prompt 57 — E-mail de prova social
Escreva um e-mail focado em depoimentos e resultados de
clientes de [produto]. Inclua: 3 depoimentos reais (nome,
resultado, citação) + dados agregados (X clientes, Y% de
satisfação) + CTA. Tom: humilde, baseado em evidências.

### Prompt 58 — E-mail de carência/último aviso
Crie um e-mail de "últimas horas" para [promoção/oferta]
que encerra em [prazo]. Assunto: urgência + benefício.
Corpo: lembrete + o que perde + CTA forte. Tom: honesto,
não apelativo. Inclua contador de tempo.

### Prompt 59 — E-mail pós-compra (cross-sell)
Escreva um e-mail para clientes que compraram [produto A]
oferecendo [produto B] como complemento. Contextualize:
"você comprou X, e para potencializar, recomendamos Y."
Benefício do combo + desconto exclusivo + CTA.

### Prompt 60 — E-mail de pedido de review
Crie um e-mail pedindo review/avaliação para [produto]
7 dias após a compra. Tom: grato, útil. Inclua: link direto
para avaliação, perguntas guia, isenção. Bônus: oferta de
desconto na próxima compra como agradecimento.

## Categoria 2: Copy para Anúncios (Prompts 61-70)

### Prompt 61 — Headlines magnéticas
Gere 20 headlines para anúncio de [produto/serviço].
4 tipos: 5 diretas (benefício claro), 5 curiosidade (gap),
5 dor (problema), 5 prova social (resultado).
Máximo: 10 palavras cada.

### Prompt 62 — Copy para anúncio Google Ads
Crie 3 anúncios para Google Ads sobre [produto/serviço].
Headline 1: palavra-chave + benefício. Headline 2: CTA.
Descrição: problema + solução + diferencial. Inclua
extensões de site links e chamada.

### Prompt 63 — Copy para anúncio TikTok
Crie um roteiro de anúncio para TikTok de 15-20 segundos
para [produto]. Hook nos 3 primeiros segundos. Desenvolvimento
rápido + demonstração visual. CTA nos últimos 3 segundos.
Tom: nativo da plataforma (informal, direto).

### Prompt 64 — Anúncio de remarketing
Crie 3 anúncios de remarketing para quem visitou mas não
comprou [produto]. Abordagens: 1) lembrete + reforço de
valor 2) objeção + contra-prova 3) desconto exclusivo.
Cada: 80-100 caracteres + CTA.

### Prompt 65 — Teste A/B de anúncio
Crie 5 variações de anúncio para [produto] para teste A/B.
Varie: headline, CTA, ângulo (dor vs. desejo), formato
(imagem vs. texto), oferta (desconto vs. bônus).
Especifique qual métrica medir para cada variação.

### Prompt 66 — Copy para anúncio de lead magnet
Crie um anúncio para promover [lead magnet gratuito] que
captura leads para [produto pago]. Headline: resultado
prometido. Descrição: o que leva + benefício rápido. CTA:
"Baixar grátis" ou "Quero [resultado]".

### Prompt 67 — Descrição de produto para marketplace
Escreva a descrição otimizada para [marketplace específico]
de [produto]. Título SEO-friendly + bullets de benefícios
+ descrição detalhada com palavras-chave + especificações
+ política de devolução. Inclua 5 tags relevantes.

### Prompt 68 — Copy para anúncio de webinar
Crie um anúncio para webinar gratuito sobre [tema].
Headline: resultado prometido no webinar. Descrição: o que
vai aprender + quem deve assistir + bônus para quem for
ao vivo. CTA: "Garantir vaga gratuita".

### Prompt 69 — Testemunho / Prova social para anúncio
Transforme o depoimento abaixo em 3 versões de anúncio: 
[cole o depoimento]. Versão 1: foco no resultado. Versão 2:
foco na transformação. Versão 3: foco na credibilidade.
Cada: 50-80 caracteres para headline + 2 linhas de apoio.

### Prompt 70 — Copy para anúncio de vagas limitadas
Crie um anúncio de escassez para [serviço/mentoria] com
[XX] vagas restantes. Headline: urgência. Descrição: o que
perde se não agir + resultado de quem já comprou + garantia.
CTA: "Últimas vagas — Garanta a sua".

## Categoria 3: Conteúdo Avançado (Prompts 71-80)

### Prompt 71 — Artigo de blog (SEO)
Escreva um artigo de blog de 1200 palavras sobre [tema].
Palavra-chave principal: [palavra]. Palavras-chave secundárias:
[lista]. Estrutura: H1, H2, H3, introdução, desenvolvimento,
conclusão, FAQ. Tom: autoritário, educativo. Meta description:
155 caracteres. Inclua 3 links internos sugeridos.

### Prompt 72 — Roteiro de vídeo de 10 minutos
Crie um roteiro de vídeo de 10 minutos para YouTube sobre
[tema]. Timeline: intro (0-1min), contexto (1-3min),
ensinamento principal (3-7min), aplicação prática (7-9min),
CTA (9-10min). Inclua: falas, imagens sugeridas na tela,
transições. Tom: didático, energético.

### Prompt 73 — E-book de 30 páginas (estrutura)
Crie a estrutura completa de um e-book de 30 páginas sobre
[tema]. Capa + sumário + 8 capítulos (cada: título, 3
subtópicos, 1 exercício) + conclusão + sobre o autor + CTA.
Para cada capítulo: objetivo de aprendizado.

### Prompt 74 — Post para LinkedIn (autoridade)
Escreva um post para LinkedIn posicionando [meu nome/marca]
como autoridade em [tema]. Hook: insight contraintuitivo
ou dado impactante. Desenvolvimento: experiência pessoal +
aprendizado + aplicação. CTA: convite para discussão.
Tom: vulnerável, confiante.

### Prompt 75 — Case de sucesso (formato história)
Escreva um case de sucesso no formato: antes (situação
problema) → processo (como resolveu) → depois (resultado
específico) → lições aprendidas. Cliente: [descreva].
Produto: [produto]. Resultado: [X]. Tom: narrativo, real.

### Prompt 76 — Guia rápido em PDF (1 página)
Crie um guia rápido de 1 página em tópicos sobre [tema].
Formato: título + subtítulo + 5-7 passos numerados + dica
de ouro + CTA. Linguagem: direta, acionável. Objetivo:
entregar valor imediato em <5 min de leitura.

### Prompt 77 — Copy para página de vendas de alto ticket
Crie a copy completa para página de vendas de [produto de
alto ticket] (R$500+). Estrutura: headline + subheadline +
vídeo/apresentação + problema agravado + solução + o que
inclui + bônus + prova social + garantia estendida + FAQ +
CTA + PS. Tom: consultivo, premium. Extensão: 2000-3000
palavras.

### Prompt 78 — Post para Instagram (carrossel educativo)
Crie um carrossel de 5 slides para Instagram sobre [tema].
Slide 1: capa atrativa (título). Slide 2: problema. Slide 3:
dados/estatística. Slide 4: solução passo a passo. Slide 5:
CTA. Inclua legenda para cada slide + texto alternativo.

### Prompt 79 — Live script (venda ao vivo)
Crie um roteiro de live de vendas de 30 minutos para
[produto]. Bloco 1 (0-5min): conexão + agenda. Bloco 2
(5-15min): conteúdo de valor. Bloco 3 (15-25min): oferta +
demonstração. Bloco 4 (25-30min): respostas + CTA final.
Inclua: frases de transição e momentos de interação.

### Prompt 80 — Copy para sequência de WhatsApp
Crie uma sequência de 7 mensagens de WhatsApp para follow-up
de leads de [produto]. Dia 1: apresentação + valor. Dia 2:
conteúdo útil. Dia 3: case. Dia 4: objeções. Dia 5: oferta.
Dia 6: urgência. Dia 7: fechamento. Cada: máximo 200
caracteres. Tom: consultivo, não invasivo.

## Categoria 4: Planejamento Estratégico (Prompts 81-90)

### Prompt 81 — Plano de marketing trimestral
Crie um plano de marketing trimestral para [negócio] com
orçamento de [R$X]. Mês 1: atração (conteúdo + tráfego).
Mês 2: conversão (ofertas + nutrição). Mês 3: retenção
(pós-venda + indicação). Para cada: canais, ações, KPI.
Produto principal: [produto].

### Prompt 82 — Funil de vendas completo
Desenhe um funil de vendas para [produto] de [preço] com:
Topo (atração): [canais + conteúdo]. Meio (nutrição):
[sequência + materiais]. Fundo (conversão): [oferta +
garantia]. Pós-venda: [sucesso do cliente + upsell].
Taxas de conversão realistas para cada etapa.

### Prompt 83 — Análise SWOT do negócio
Faça uma análise SWOT para [meu negócio/nicho]. Forças
(internas): [liste]. Fraquezas (internas): [liste].
Oportunidades (externas): [liste]. Ameaças (externas):
[liste]. Para cada: impacto (alto/médio/baixo) + ação
recomendada.

### Prompt 84 — Matriz de conteúdo
Crie uma matriz de conteúdo para [nicho] com 20 ideias
distribuídas em: formato (post/blog/vídeo/podcast) x
estágio do funil (topo/meio/fundo). Cada ideia: título +
formato + objetivo. Nicho: [descreva].

### Prompt 85 — Plano de lançamento em 30 dias
Crie um plano de lançamento de 30 dias para [produto].
Fase 1 (D1-7): pesquisa + criação. Fase 2 (D8-14):
pré-lançamento + aquecimento. Fase 3 (D15-21): lançamento
+ vendas. Fase 4 (D22-30): pós-lançamento + otimização.
Para cada fase: ações diárias + metas.

### Prompt 86 — Sistema de indicadores (dashboard)
Crie um dashboard de indicadores para [negócio] com:
Financeiro (receita, margem, ticket médio). Marketing
(leads, CPL, conversão). Vendas (vendas/dia, taxa de
fechamento). Produto (NPS, churn, LTV). Frequência de
medição: semanal/mensal. Metas para cada.

### Prompt 87 — Pesquisa de mercado
Crie uma pesquisa de 10 perguntas para validar [ideia de
produto] com [público-alvo]. Perguntas: 3 demográficas,
4 comportamentais, 3 de intenção de compra. Inclua escala
de 1-5 e perguntas abertas. Público-alvo: [descreva].

### Prompt 88 — Roteiro de entrevista com cliente
Crie 12 perguntas para entrevistar [cliente ideal] sobre
[dores e desejos]. Perguntas abertas: rotina, frustrações,
tentativas anteriores, orçamento, critérios de decisão,
resultado desejado. Objetivo: entender a fundo a jornada
do cliente. Duração: 20-30 min.

### Prompt 89 — Análise de concorrência
Analise [concorrente 1] e [concorrente 2] em: produto,
preço, posicionamento, canais, público, diferenciais,
fraquezas. Me dê: tabela comparativa + gaps de mercado
(oportunidades não exploradas) + recomendações para me
diferenciar.

### Prompt 90 — Plano de ação 30-60-90 dias
Crie um plano de ação de 90 dias para [objetivo específico].
Dias 1-30: fundação (o que construir). Dias 31-60:
crescimento (o que escalar). Dias 61-90: otimização (o que
refinar). Para cada período: 3 tarefas principais + 1 KPI.
Objetivo: [meta específica].

## Categoria 5: Bônus Avançados (Prompts 91-100)

### Prompt 91 — Script de call de vendas completa
Crie um script completo de call de vendas de 20 min para
[produto] de [preço]. Abertura (2min): rapport. Descoberta
(5min): perguntas abertas. Apresentação (5min): solução
personalizada. Objeções (3min): respostas. Fechamento
(3min): assumir a venda. Pós (2min): próximos passos.
Para cada bloco: falas literais + perguntas-chave.

### Prompt 92 — Proposta comercial (página)
Crie a estrutura de uma proposta comercial de 1 página para
[produto/serviço]. Seções: entendimento do cliente, solução
proposta, resultados esperados, investimento, condições,
próximos passos. Tom: consultivo, profissional. Inclua
placeholder para logo e dados do cliente.

### Prompt 93 — Contrato simples de prestação de serviço
Esboce um contrato simples de prestação de serviço digital
para [tipo de serviço]. Cláusulas: objeto, prazo, valor,
forma de pagamento, entregas, confidencialidade, rescisão,
foro. Observação: não substitui advogado. Linguagem: clara,
acessível.

### Prompt 94 — Pesquisa de NPS
Crie uma pesquisa de NPS (Net Promoter Score) para enviar
30 dias após a compra. Pergunta principal: "De 0 a 10, o
quanto você recomendaria [produto]?" + pergunta aberta de
justificativa + pergunta de melhoria. Inclua: introdução
acolhedora e agradecimento.

### Prompt 95 — Sequência de upsell
Crie uma oferta de upsell para cliente que comprou [produto
entry] no valor de [R$X]. Oferta: [produto PRO] por [R$Y]
(desconto de [Z]% para compradores). Gatilho: logo após a
compra inicial. Copy: "você comprou X, e para [resultado
maior], adicione Y por apenas mais [valor]."

### Prompt 96 — Estratégia de indicação
Crie um programa de indicação para [produto/serviço].
Mecânica: [cliente indica X amigos] e ganha [benefício].
Copy para: 1) convite de indicação 2) lembrete 3)
agradecimento. Benefício para quem indica e para quem é
indicado. Regras claras em 1 parágrafo.

### Prompt 97 — Script de agradecimento ao afiliado
Crie 3 mensagens de agradecimento para afiliado que vendeu
[produto]: 1) 1a venda (celebração) 2) 5a venda (reconheci-
mento) 3) 10a venda (bônus surpresa). Inclua: dados da
venda, elogio específico, incentivo para continuar.

### Prompt 98 — Briefing para designer
Crie um briefing completo para designer criar [material:
logotipo/landing page/apresentação]. Inclua: referências
visuais, paleta de cores, tipografia, tom, exemplos de
concorrentes, público-alvo, formato de entrega. Objetivo
[descrição]. Prazo: [X dias].

### Prompt 99 — Roteiro de apresentação de pitch
Crie um pitch de 3 minutos para [produto/negócio].
Minuto 1: problema + solução. Minuto 2: mercado + tração.
Minuto 3: time + pedido. Para cada: 1-2 frases-chave.
Objetivo: [conseguir investimento/parceiro/cliente].

### Prompt 100 — Plano de desenvolvimento pessoal
Crie um plano de 30 dias para desenvolver [habilidade
específica] com: Semana 1: fundamentos (o que estudar).
Semana 2: prática guiada (exercícios). Semana 3: aplicação
real (projeto). Semana 4: ensino (consolidar). Inclua
recursos gratuitos recomendados e métrica de progresso.
